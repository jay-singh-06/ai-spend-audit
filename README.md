# AI Spend Audit Optimizer

An AI-powered SaaS application that analyzes organizational AI tooling spend and identifies potential cost optimization opportunities.

---

# Features

- Multi-tool AI spend analysis
- AI-generated optimization summary
- Savings estimation
- PDF report generation
- Supabase database integration
- Dynamic multi-tool input system
- Spam protection using honeypot field
- CI workflow using GitHub Actions

---

# Supported Tools

- ChatGPT
- Claude
- GitHub Copilot
- Cursor
- Gemini
- OpenAI API
- Anthropic API
- Windsurf
